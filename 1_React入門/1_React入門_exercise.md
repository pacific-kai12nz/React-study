## 問題2：JSXの基本
以下のHTMLを、JSXを使って書き直してください。

```html
<div class="container">
  <h1>Shopping List</h1>
  <ul>
    <li>Apple</li>
    <li>Banana</li>
    <li>Orange</li>
  </ul>
</div>
```

JSXでの書き方：
```jsx
// ここにJSXコードを記述してください
function ShoppingList() {
  return (
    <div className="container">
      <h1>Shopping List</h1>
      <ul>
        <li>Apple</li>
        <li>Banana</li>
        <li>Orange</li>
      </ul>
    </div>
  );
}
```
